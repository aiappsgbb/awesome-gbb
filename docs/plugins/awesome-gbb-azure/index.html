awesome-gbb-azure

v1.0.0 18 skills

Microsoft GBB skills for Azure + Microsoft Foundry pilots. Bundles 18 production-tested skills across 5 areas: (1) Foundry building blocks — hosted agents, RAG (Foundry IQ), MCP servers on ACA, evaluation, observability, vision/speech/doc, Teams bot, toolbox, skill catalog, cross-resource, VNet-isolated; plus (2) ghcp-hosted-agents for using GitHub Copilot models from Foundry; (3) Azure infrastructure — azd-patterns (Bicep module library + ACA job patterns), azure-tenant-isolation (multi-tenant az/azd safety); (4) Governance — citadel-hub-deploy (APIM AI Gateway), citadel-spoke-onboarding, foundry-agt (Agent Governance Toolkit); (5) Cost — paygo-ptu-cost-analyzer (PayGo vs PTU sizing reports). USE FOR: any Foundry pilot, AOAI capacity planning, multi-tenant azd workflow, agent governance, APIM gateway. DO NOT USE FOR: Threadlight engagements (use awesome-gbb-threadlight — bundles these plus the threadlight-* pipeline).

Install

copilot plugin marketplace add aiappsgbb/awesome-gbb
copilot plugin install awesome-gbb-azure@awesome-gbb
