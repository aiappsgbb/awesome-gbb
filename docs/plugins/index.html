Plugins

Three Copilot CLI plugin bundles that install whole engagement domains in one command. Skills also remain installable individually via gh skill install.

Register the marketplace

copilot plugin marketplace add aiappsgbb/awesome-gbb

awesome-gbb-azure

Microsoft GBB skills for Azure + Microsoft Foundry pilots.

copilot plugin install awesome-gbb-azure@awesome-gbb
18 skills v1.0.0

awesome-gbb-basic

Cross-cutting Microsoft GBB skills usable in any pre-sales or pilot engagement, regardless of cloud or stack.

copilot plugin install awesome-gbb-basic@awesome-gbb
7 skills v1.0.0

awesome-gbb-threadlight

End-to-end Threadlight engagement pipeline as a single self-contained plugin.

copilot plugin install awesome-gbb-threadlight@awesome-gbb
