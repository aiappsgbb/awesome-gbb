awesome-gbb-basic

v1.0.0 7 skills

Cross-cutting Microsoft GBB skills usable in any pre-sales or pilot engagement, regardless of cloud or stack. Bundles: research-company (web research → org brief), compose-org (fork an agentic substrate into a customer-flavoured digital clone), gbb-humanizer (polish prose with the GBB voice), gbb-pptx (Office-quality deck generation), auto-demo-producer (Playwright + edge-tts + ffmpeg narrated demos), ip-catalog (shared IP discovery), ghcp-cli-config (bootstrap Copilot CLI with 6 recommended MCP servers + settings + plugins). USE FOR: any engagement that needs research, prose polish, demo video production, deck authoring, or Copilot CLI baseline setup. DO NOT USE FOR: Azure or Foundry work (use awesome-gbb-azure) or Threadlight pipelines (use awesome-gbb-threadlight).

Install

copilot plugin marketplace add aiappsgbb/awesome-gbb
copilot plugin install awesome-gbb-basic@awesome-gbb
