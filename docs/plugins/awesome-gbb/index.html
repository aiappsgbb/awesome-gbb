awesome-gbb

v2.0.0 36 skills

Microsoft GBB skills catalog — 38 reusable building blocks for Copilot-assisted engagements. Covers seller enablement, Azure infra (azd, Bicep, ACA), Foundry agents (hosted agents, RAG, MCP, evals, observability), governance (Citadel, AGT), Threadlight end-to-end pipeline, digital-clone orchestration, cost analysis, tenant isolation, and enterprise automation. USE FOR: any pre-sales engagement, pilot, PoC, or demo that involves Azure AI, Foundry, or Copilot CLI workflows. DO NOT USE FOR: nothing — this is the full catalog.

Install

copilot plugin marketplace add aiappsgbb/awesome-gbb
copilot plugin install awesome-gbb@awesome-gbb
