awesome-gbb

v4.5.1 27 skills

Microsoft AI Apps GBB skills catalog — 28 reusable building blocks for Copilot-assisted engagements. Covers Azure infra (azd, Bicep, ACA), Foundry agents (prompt agents, hosted agents, RAG, MCP, evals, observability), governance (Citadel, AGT), cost analysis, tenant isolation, and enterprise automation. Threadlight pipeline skills moved to threadlight-skills. Zava digital-clone skills moved to zava-constellation. USE FOR: any pre-sales engagement, pilot, PoC, or demo that involves Azure AI, Foundry, or Copilot CLI workflows. DO NOT USE FOR: Threadlight pipeline (use threadlight-skills), Zava workspace (use zava-constellation).

Install

copilot plugin marketplace add aiappsgbb/awesome-gbb
copilot plugin install awesome-gbb@awesome-gbb
