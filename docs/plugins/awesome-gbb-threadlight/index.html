awesome-gbb-threadlight

v1.0.3 23 skills

End-to-end Threadlight engagement pipeline as a single self-contained plugin. Bundles 8 threadlight-* skills (design → demo-data-factory → local-test → safe-check → deploy → event-triggers → hitl-patterns → workspace-ui) PLUS the 15 Foundry/Azure/basic skills they cross-reference at runtime (foundry-hosted-agents, foundry-iq, foundry-observability, foundry-mcp-aca, foundry-evals, foundry-doc-vision-speech, foundry-teams-bot, foundry-cross-resource, ghcp-hosted-agents, azd-patterns, azure-tenant-isolation, citadel-spoke-onboarding, auto-demo-producer, gbb-humanizer, gbb-pptx). Install this plugin alone for a full Threadlight pilot — no other awesome-gbb plugins required. USE FOR: scoping a Threadlight pilot, designing a process SPEC, generating demo data, local Pattern 0/1/2/3 testing, pre-deploy safe-check gate, azd deploy, HITL + Teams + event triggers, cinematic workspace UI. DO NOT USE FOR: non-Threadlight Foundry pilots — use awesome-gbb-azure instead.

Install

copilot plugin marketplace add aiappsgbb/awesome-gbb
copilot plugin install awesome-gbb-threadlight@awesome-gbb
