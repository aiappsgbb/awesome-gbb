Get started in one command

Add the marketplace, then install whichever bundle matches the engagement — three domains, zero auth gymnastics.

copilot plugin marketplace add aiappsgbb/awesome-gbb
copilot plugin install awesome-gbb-basic@awesome-gbb       # universal skills
copilot plugin install awesome-gbb-azure@awesome-gbb       # + Azure / Foundry / governance
copilot plugin install awesome-gbb-threadlight@awesome-gbb # + end-to-end pilot pipeline

Or pick one skill at a time: gh skill install aiappsgbb/awesome-gbb foundry-iq
