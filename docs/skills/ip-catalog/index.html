ip-catalog

v1.0.0 🔍 Discovery

Discover AI Apps GBB IP catalog via MCP server. Search, list, filter, and inspect IPs (intellectual property assets) including metadata, thumbnails, and READMEs. USE FOR: find IP, search demos, list IPs, catalog, what demos are available, find solutions by service/pattern, get IP details, browse catalog, discover assets. DO NOT USE FOR: creating IPs, deploying apps, managing repos.

Install

gh skill install aiappsgbb/awesome-gbb ip-catalog
