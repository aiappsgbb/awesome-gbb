zava-workspace-deploy

v1.0.0 🧬 Org Composition

Deploy custom React/Vite SPAs to Azure Container Apps as static-serve containers. Builds the SPA, packages with nginx, generates Bicep + azure.yaml service entry. Handles API proxy to the Foundry-hosted control plane. Reusable for any custom web UI beyond threadlight-workspace-ui's generated reference UIs. USE FOR: deploy React app, deploy Vite SPA, static site on ACA, custom workspace deploy, Zava control plane UI, candidate portal, blueprint microsite, custom dashboard, nginx container, SPA hosting Azure. DO NOT USE FOR: generated threadlight workspace UIs (use threadlight-workspace-ui), full-stack app deployment (use threadlight-deploy), static marketing sites (use Azure Static Web Apps directly).

Install

gh skill install aiappsgbb/awesome-gbb zava-workspace-deploy
